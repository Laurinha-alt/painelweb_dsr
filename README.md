Tecnologias Usadas:
Aqui estão as principais tecnologias utilizadas no projeto:

React: É o "cérebro" do site. Em vez de escrever código misturado, a gente monta o site como se fossem blocos de LEGO (componentes reutilizáveis).

Vite: É o "motor". Ele faz o site abrir instantaneamente no seu computador enquanto trabalhamos nele.

Tailwind CSS: É o "estilista". Ele deixa o site bonito, alinhado e responsivo (funciona no celular) sem precisarmos escrever arquivos CSS gigantes.

Leaflet: É o "cartógrafo". É a biblioteca que desenha o mapa na tela, entende coordenadas e nos permite colocar as camadas exportadas do QGIS.

Lucide React: São os ícones modernos (lupa, menu, download) que usamos na interface.

Comandos Úteis:

npm install: Instala as dependências do projeto.

npm run dev: Inicia o servidor local para desenvolvimento.

npm run build: Cria a versão final otimizada para colocar no ar.
